给强迫症一个干净整洁的 Adobe Acrobat

强迫症的碎碎念

多年以来, 笔者一直在寻找一个 Windows 上的足够完美的 PDF 阅读器, 但是苦于各个阅读器都有各种各样的缺点, 使用的时候总会觉得心神不宁, 坐立难安. 下文中将首先罗列一下笔者使用的 PDF 阅读器的特典, 只对 Acrobat 感兴趣的读者可以跳过到下一节.

Sumatra PDF

从功能上来说, Sumatra 几乎是完美的阅读器, 轻量又快速, 不会锁定 PDF 文件, 还支持 VIM 键位. 但非常可惜的是 UI 不好看. 尽管在某一个版本把工具栏上旧的拟物图标换成了简洁线条风格的图标, 但是主页上的不可用的工具栏图标仍然很难看, 因此笔者暂时不会把它作为平时主力使用的 PDF 阅读器.

但是 “不锁定 PDF 文件” 的功能还是很重要的, 而且这还是 Sumatra 具有但 Acrobat 没有的功能, 所以在编写 LaTeX 文档的时候, 笔者还是会使用 Sumatra 用于查看编译结果.

福昕

Foxit 的 PDF 阅读器和编辑器的版本数不胜数, 而且还分国内版和海外版, 这里所指的是国内版的 “福昕高级PDF编辑器 专业版”.

Foxit 给我的第一印象就是很差的, 因为他们的首页就非常土, 软件的图标也很难看. 使用之后的感受是, 功能足够全面, 但是 OCR 速度明显不如 Acrobat, UI 只能说称不上难看.

万兴 PDF 阅读器

和 Foxit 有一样的毛病, 版本很多. UI 虽然现代但也很土, 功能也比较全面, 但是性能比较差, 启动很慢, 用着也不顺手.

Acrobat

功能全面, 性能也是最好的. 缺点不少, 但都是可以解决的. 具体的缺点如下:

  • 在中国大陆不向个人用户销售正版. 这很好, 心安理得地用盗版了.
  • 内存占用远高于 Sumatra. 不过关闭 “页面显示 - 渲染 - 使用页面高速缓存” 之后占用的内存还是在可以接受的范围内的.
  • UI 上有很多烦人的贴片广告, 以及没有必要的在线账户功能. 下一节就是介绍如何关闭这些东西.
    也就是说, 只要能去掉 UI 上的一些部分, 那么 Acrobat 就能满足笔者的需要. 事实上这些都是可以设置的, 只是 Adobe 没有将这些设置项展示出来.

如何关闭 Acrobat 界面上的小广告

Acrobat 有大量的未在 “首选项” 中展示的设置选项, 它们都保存在 Windows 的注册表中. 我们可以直接修改注册表, 也可以使用 Adobe Customization Wizard 来修改. 本文只介绍前一种方法, 专为这几个设置项下载一个软件是没有必要的.

下文中提到的注册表中的值, 全部都是 类型. 如果注册表中不存在这些值, 直接新建一个即可.

  • 隐藏标签栏中的登录入口: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\<product name>\<version>\FeatureLockDown\cServices\bUpdater 设置为 0 (DWORD 32bit).

  • 隐藏 “工具栏 - 帮助” 中的登录入口: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\<product name>\<version>\FeatureLockDown\bSuppressSignOut 设置为 0 (DWORD 32bit).

  • 隐藏主页第一行中 app 的广告等: Computer\HKEY_CURRENT_USER\Software\Adobe\<product name>\<version>\HomeWelcomeFirstMile\bEnableFirstMile 设置为 0 (DWORD 32bit).

  • 隐藏文档阅读界面右下角的 store and share files in the document cloud: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Adobe\<product name>\<version>\FeatureLockDown\cIPM\bDontShowMsgWhenViewingDoc 设置为 0 (DWORD 32bit).

对于 Acrobat DC 来说, <product name> 处应填 “Adobe Acrobat”, <version> 处应填 “DC”. 这些设置也适用于 Adobe Reader.

2023.3.22 更新

笔者发现 PDF-XChange Editor 实际上是更好的选择:

  • 它的免费版功能足够全面, 而且可以在设置中关闭所有付费功能的入口;
  • 启动快, 性能好, 占用内存远少于 Acrobat (但还是比 Sumatra 更多);
  • 外观虽然不大好看, 但是用户对 UI 的控制很强, 可以定制出符合自己习惯的 UI.

所以笔者目前的 PDF 解决方案是, 论文使用 Zotero 内置的阅读器, 这样很方便同步并管理批注; 书本和其它文件就用 PDF-XChange Editor.
