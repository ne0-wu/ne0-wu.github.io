有理曲线

在前几篇文章中, 我们已经介绍了 Bézier 曲线和样条曲线, 它们分别是多项式曲线和分段多项式曲线, 其数学意义在于构造可以任意地逼近任何光滑曲线的参数曲线. 但是我们并不满足于是逼近, 我们希望精确地表示更多种类的曲线, 如圆锥曲线.

圆锥曲线与有理函数

以最简单的一类圆锥曲线为例: 二维平面中的单位圆 x2+y2=1x^2+y^2 = 1x2+y2=1, 很容易验证它不能被任何多项式精确表示. 但是可以用三角函数来将其参数化:

x=cos⁡θ, y=sin⁡θ, θ∈[0,2π) x = \cos \theta,~ y = \sin \theta,~ \theta \in [0, 2\pi) x=cosθ, y=sinθ, θ∈[0,2π)

考虑三角函数的万能公式

cos⁡θ=1−tan⁡2θ21+tan⁡2θ2, sin⁡θ=2tan⁡θ21+tan⁡2θ2 \cos \theta = \frac{1 - \tan ^2 \frac{\theta}{2}}{1 + \tan ^2 \frac{\theta}{2}},~ \sin \theta = \frac{2 \tan \frac{\theta}{2}}{1 + \tan ^2 \frac{\theta}{2}} cosθ=1+tan22θ​1−tan22θ​​, sinθ=1+tan22θ​2tan2θ​​

令 t=tan⁡θ2t = \tan \frac{\theta}{2}t=tan2θ​, 可以得到圆的另一个参数化

x=1−t21+t2, y=2t1+t2, t∈(−∞,∞) x = \frac{1-t^2}{1+t^2},~ y = \frac{2t}{1+t^2},~ t \in (-\infty, \infty) x=1+t21−t2​, y=1+t22t​, t∈(−∞,∞)

形如这样的由多项式除法的商得到的函数被我们称为有理函数 (Rational Function), 有理曲线就是利用有理函数构造的曲线.

不难注意到, 其它的圆锥曲线也可以通过三角函数或双曲函数及对应的万能公式用有理函数来表示, 因此有理曲线包含了多项式曲线和圆锥曲线.

除法的意义

本节中我们将讨论上一节中, 有理函数中除法的意义, 以及如何理解有理函数与圆锥曲线之间的关系.

仍然以二维平面中的单位圆为例, 考虑三维空间中的多项式曲线

x=1−t2, y=2t, z=1+t2, t∈(−∞,∞) x = 1-t^2,~ y = 2t,~ z = 1+t^2,~ t \in (-\infty, \infty) x=1−t2, y=2t, z=1+t2, t∈(−∞,∞)

以原点为相机, 在平面 z=1z=1z=1 上做透视投影, 得到

x=1−t21+t2, y=2t1+t2, z=1, t∈(−∞,∞) x = \frac{1-t^2}{1+t^2},~ y = \frac{2t}{1+t^2},~ z = 1,~ t \in (-\infty, \infty) x=1+t21−t2​, y=1+t22t​, z=1, t∈(−∞,∞)

发现单位圆就是三维空间中多项式曲线在二维平面上的投影. 类似地, 这也适用于任何其它的有理曲线.
